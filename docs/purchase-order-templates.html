Purchase Order Template for Excel
Procurement uses the Excel purchase order template for ordering supplies or materials. Includes second page for added procurement detail along with third tab for receiving items ordered.

Customize Purchase Order Template
Add your company name and logo to customize the purchase order template. Resave the template. Select file, save as, select xltx, rename the template name if desired.

Organizing Purchase Orders
Create a folder name "my purchase orders", create subfolders named draft, open, cancelled and closed. Track status of each purchase order by savings file in the appropriate folder.
